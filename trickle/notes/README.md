# OCTAHIRE - Campus Recruitment System

## Overview
OCTAHIRE is a comprehensive campus recruitment platform developed by Team8. It connects universities, students, and companies on a single platform to streamline the hiring process.

## Key Features
- **Student Management**: Profile management and application tracking
- **Company Portal**: Job posting, candidate review, and interview scheduling
- **University Dashboard**: Student database management and placement statistics
- **Smart Scheduling**: Automated interview scheduling with conflict detection
- **Analytics**: Real-time insights and placement reports
- **Communication Hub**: Integrated messaging system for all users

## Pages
- `index.html`: Landing page with features and how it works
- `login.html`: Interactive login page with role selection
- `signup.html`: Interactive signup page with validation
- `dashboard.html`: Main dashboard with role-based views
- `user-manual.html`: Complete user guide and documentation

## User Types
1. **Students**: Browse jobs, apply to positions, track application status
2. **Companies**: Post jobs, review candidates, schedule interviews, select candidates
3. **Universities**: Manage students, coordinate placements, monitor statistics, generate reports
